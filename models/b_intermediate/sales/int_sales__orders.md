{% docs int_sales__orders %}

This model provides an aggregated view of orders, combining data from multiple sources such as order items, user details, staffs and stores details. It enriches the order data with the following metrics:
- **Total Items**: The total quantity of items in the order.
- **Total Gross Revenue**: Sum of gross revenue in the order.
- **Total Net Revenue**: Sum of net revenue, including discount.

It provides a comprehensive view of each order, allowing for easy analysis of order performance.
